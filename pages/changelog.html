v0.0.1 Initial Release
January 19, 2026
Added
  • Project Setup — Initial repository structure and open source standards.
  • PDF Editing Core — Feature-rich PDF engine based on PDF.js and pdf-lib.
  • Modern Toolbar — Floating toolbar with tool groups (General, Edit, Annotate, Sign, Actions).
  • Text Tools — Add new text, edit existing PDF.
  • Drawing — Freehand drawing tool with multiple colors and stroke widths.
  • Shapes — Rectangle, Circle, Line, and Arrow tools with style customization.
  • Images — Drag & drop or upload images directly into the PDF.
  • Signatures — Create, save, and place signatures and stamps.
  • Annotations — Highlight tool and comment/sticky note functionality.
  • Page Management — Reorder, rotate, and delete pages with visual grid.
  • Security — Password protection, watermarking, and permanent redaction.
  • Privacy-First — All PDF processing happens locally in the browser.
